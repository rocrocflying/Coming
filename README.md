#Coming
本程序基于web开发，主要包括约吧、淘吧、吃吧三大模块，主要为昆明呈贡大学城在校师生提供方便、快捷、贴心的生活娱乐服务。
其中：
约吧主要功能有表白墙、老乡会、组件活动。
淘吧主要功能有二手跳蚤市场、拍卖商品。
吃吧主要功能有美食墙、推荐美食模块、约吃。
